# 📘 DBMS Lab + JDBC Projects

A comprehensive collection of **DBMS lab exercises** along with **Java JDBC-based mini projects** demonstrating real-world database applications.

---

## 📂 Repository Structure

### 🔹 SQL Scripts (Core DBMS Concepts)

| File | Description |
|------|------------|
| `ddl.sql` | Table creation and schema design |
| `dml.sql` | Insert, update, delete operations |
| `dql.sql` | Select queries |
| `setsandjoins.sql` | Joins and set operations |
| `Subqueries.sql` | Nested queries |
| `IndexandView.sql` | Views and indexing |
| `Triggers.sql` | Database triggers |
| `FunctionandProcedure.sql` | Stored procedures & functions |
| `Plsql.sql` | PL/SQL programs |

---

### 🔹 JDBC Projects (Java + Database Integration)

Located in `JDBC/` folder:

| File | Description |
|------|------------|
| `EmployeeManagement.java` | GUI-based employee management system |
| `EmployeeManagementConsole.java` | Console-based CRUD operations |
| `ShopManagementSystem.java` | Full-featured shop management system |

---

## 🎯 Objectives

- Learn **DBMS fundamentals**
- Practice **SQL queries and database design**
- Understand **JDBC connectivity**
- Build **real-world database applications**
- Implement **CRUD operations in Java**
